# 📦 نظام إدارة الحسابات - (Cloud Edition)

نظام محاسبي ذكي يعتمد على تقنيات الويب لإدارة سجلات العملاء وعمليات المصارفة اليومية بسهولة وأمان.

## ✨ المميزات:
- **إدارة العملاء:** إضافة وتتبع سجلات العملاء الجدد.
- **نظام المصارفة:** تسجيل العمليات المالية (ريال سعودي، ريال يمني، دولار).
- **واجهة مستخدم عصرية:** متوافقة تماماً مع أجهزة الجوال والمتصفحات.
- **قاعدة بيانات سحابية:** حفظ البيانات بشكل دائم باستخدام SQLite.

## 🛠 التقنيات المستخدمة:
- **اللغة:** Python 3
- **إطار العمل:** Flask (Web Framework)
- **قاعدة البيانات:** SQLite3
- **التصميم:** HTML5 & CSS3 (Responsive Design)

## 🚀 التشغيل:
المشروع جاهز للاستضافة على منصات مثل Render أو PythonAnywhere.
